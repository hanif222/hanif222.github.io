TEXTBOX

Membuat sebuah textbox cukup mudah. Kita tinggal membuat sebuah object. Lalu menambahkan beberapa event berikut.

CREATE EVENT

textbox0 = hanif_textbox_create(32, 32, 32 * 18, -1); //-1 untuk default
hanif_textbox_set_label(textbox0, "Masukkan nama anda");

nama = "";

STEP EVENT

hanif_textbox_step(textbox0);

nama = hanif_textbox_get_value(textbox0);

DRAW EVENT

draw_text(0, 0, "Nama anda adalah: " + nama);

DRAW GUI EVENT

hanif_textbox_draw(textbox0);

PASSWORD

Menjadikan textbox ke bentuk password cukup mudah. Kita tinggal memanggil fungsi berikut.

hanif_textbox_password(textbox0, true);
